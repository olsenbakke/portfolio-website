# Portfolio Website

A clean, responsive developer portfolio website built with HTML, CSS, and JavaScript. This starter project is designed to be easy to customize and deploy.

## Features
- Responsive layout
- Hero, about, projects, and contact sections
- Smooth scrolling navigation
- Simple project cards
- Easy customization

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript

## Getting Started
1. Open `index.html` in your browser.
2. Customize the content in `index.html`.
3. Update colors and layout in `styles.css`.
4. Edit interactions in `script.js`.

## Project Structure
```
portfolio-website/
├── index.html
├── styles.css
├── script.js
└── README.md
```
